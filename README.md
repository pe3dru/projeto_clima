# Projeto Clima

## O que é esse projeto?

O **Projeto Clima** é um site simples e prático onde você pode visualizar a **previsão do tempo** da sua cidade.  
Ele exibe informações como **temperatura** e **condições climáticas** (por exemplo: ensolarado, nublado ou chuvoso).

A ideia é permitir que qualquer pessoa digite o nome de uma cidade e, em poucos segundos, veja o clima atual de forma clara e acessível.

---

## Tecnologias utilizadas

Mesmo que você não conheça programação, aqui vai uma explicação simples das ferramentas usadas:

- **HTML** → É a estrutura do site (o que aparece na tela).  
- **CSS** → Cuida da aparência (cores, tamanhos, espaçamento, fontes, etc.).  
- **JavaScript** → É o cérebro do site, responsável por buscar e exibir as informações do clima.  
- **Node.js e npm** → Usados para instalar e organizar os arquivos do projeto.  
- **API de Clima (OpenWeather)** → Site externo que se conecta com nossa aplicação para fornecer e exibir os dados da cidade que você procurar.

*Resumindo:*  
O **HTML** é o **esqueleto**, o **CSS** é a **roupa**, e o **JavaScript** é o **cérebro** que dá vida ao site.

## Como rodar o projeto

Mesmo que você nunca tenha feito isso antes, siga o passo a passo com calma, é simples!

### 1. Baixar o projeto
Clique no botão **Code > HTTPS** e copie o link do repositório para seu computador.

### 2. Depois, abra o terminal e digite:
git clone https://github.com/pe3dru/projeto_clima.git

### 3. Depois, entre na pasta do projeto:
cd projeto_clima

O projeto usa algumas “bibliotecas” que são como ferramentas auxiliares. Para instalá-las, é preciso ter o Node.js instalado no seu computador.
### 4. Para instalar o node insira em seu terminal:
npm init -y

### 5. Depois do node instalado, abra o terminal (ou prompt de comando) e digite:
npm install
### Isso vai preparar o ambiente para o site funcionar direitinho.

### 6. Agora é hora de ver o projeto funcionando! No terminal, digite:
npm start

### 7. Em seguida, abra o navegador e acesse:
http://localhost:3000

Você verá o site funcionando na sua própria máquina.

## Como rodar o projeto
1. Digite o nome da cidade desejada (por exemplo: São Jose da Coroa Grande).
2. Clique no botão de buscar.
3. O site mostrará:
   I. Temperatura atual
   II. Condição do tempo (sol, chuva, nublado etc)
4. Aproveite para testar outras cidades também!
